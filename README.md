# 🌍 Language Translation Tool

## 📌 Project Overview
This project is developed as part of the CodeAlpha Artificial Intelligence Internship. The Language Translation Tool allows users to translate text from one language to another using the Google Translator API through the deep_translator library.

## 🚀 Features
- Translate text between multiple languages
- User-friendly command-line interface
- Fast and accurate translations
- Supports language codes such as English (en), Telugu (te), Hindi (hi), French (fr), etc.

## 🛠️ Technologies Used
- Python
- deep_translator
- Google Translator API

## 📂 Project Structure

## ▶️ How to Run

1. Install the required library:

pip install deep-translator

2. Run the program:

python translation_tool.py

3. Enter:
- Text to translate
- Source language code
- Target language code

4. View the translated output.

## 📝 Sample Input

Enter text: Hello

Source language (e.g., en): en

Target language (e.g., hi): hi

## 📝 Sample Output

Translated Text:

नमस्ते
